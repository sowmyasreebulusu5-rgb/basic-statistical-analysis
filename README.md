# Basic Statistical Analysis of Student Marks Using Pandas

## Objective
This project performs basic statistical analysis on a dataset of student marks using Python and pandas.

## Files in this project
- student_marks.csv → dataset
- basic_statistical_analysis.py → Python code
- README.md → project explanation

## Statistical Measures Calculated
- Count
- Mean
- Median
- Mode
- Minimum
- Maximum
- Range
- Variance
- Standard Deviation

## Conclusion
This project helped me understand how pandas can be used to load and analyze a dataset.
